Папка для препроцессорных файлов разметки страниц.

--

Описание структуры файлов:

template/
  ├── instruction/
    ├── _favicon.html
    └── другие _*.html файлы содержащие служебную информацию
  ├── section/
    ├── _header.html
    └── другие _*.html файлы размечающие секцию
  ├── stencil/
    ├── _default.html
    └── другие _*.html файлы содержащие общий шаблон
  ├── main.html
  ├── другие *.html файлы размечающие основу страницы
  └── readme.md подробно описывает структуру препроцессорных файлов для шаблонизатора страниц и иные правила 

--

Пояснениение именования файлов:

* если файл начинается с _ значит он содержит только часть разметки,
* если файл не начинается с _ значит это страница которую нужно собрать.
